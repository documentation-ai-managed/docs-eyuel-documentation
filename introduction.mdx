## Overview

Eyuel Documentation provides a centralized space to create, organize, and manage all your project documentation. You build comprehensive guides using MDX, leveraging powerful components for interactive and visually rich pages. Customize your docs with themes, search, and navigation to help teams collaborate effectively.

## Get Started in Minutes

Follow these steps to launch your first documentation site.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://app.eyuel.com` and verify your email.
  </Step>
  <Step title="Initialize Project" icon="folder-plus">
    Create a new repo and add your first MDX file.

    ```bash
    mkdir my-docs && cd my-docs
    npm init -y
    npm install @eyuel/docs
    ```
  </Step>
  <Step title="Configure & Deploy" icon="rocket">
    Edit `eyuel.config.js` and push to deploy.

````javascript
// eyuel.config.js
module.exports = {
  title: 'My Project Docs',
  brandColor: '#3B82F6',
  nav: [
    { title: 'Introduction', href: '/introduction' },
    { title: 'API', href: '/api' }
  ]
};
````
  </Step>
</Steps>
